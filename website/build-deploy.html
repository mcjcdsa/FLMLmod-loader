打包与部署

本页汇总 flml-core 官方核心包的打包、验证与部署流程，以及模组的构建建议。

FLML 核心包（flml-core）

  • Maven/Gradle 配置：依赖、shade/relocation、Jar Manifest
  • 官方标识与版本校验：FLML-Official、Game-Version、API-Version
  • 游戏启动时校验：FLMLCoreValidator

参考文档

  • FLML 官方核心包打包及部署规范
  • flml-core README

模组打包建议

  • 明确 mod.json 元数据与依赖关系
  • 按需打包运行时依赖，避免与 FLML 冲突
  • 可选 JAR 签名，确保供应链安全
  • 提供 CHANGELOG 与版本检查机制

部署与验证

  1. 将 flml-core-1.0.0.jar 放入 mods/
  2. 将模组 JAR 放入 mods/
  3. 通过官方启动器启动（令牌校验），在日志中确认加载结果
