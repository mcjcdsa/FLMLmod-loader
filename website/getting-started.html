快速开始

本页指导你在 FreedomLand 中启用官方模组加载器（FLML）并创建你的第一个模组。

1. 前置条件

  • 已安装 JDK 17+
  • 已构建游戏多模块工程（Launcher / GameCore / flml-core）
  • 可选：Maven 或 Gradle 环境

2. 安装 FLML 核心包

  1. 构建或下载 flml-core-1.0.0.jar
  2. 将其放置到游戏 mods/ 目录或官方指定目录
  3. 启动器启动游戏后，GameCore 会校验 FLML 核心包有效性

详见：打包与部署

3. 创建示例模组

  1. 在 mods/ 下建立你的模组目录，包含 mod.json、资源与源码
  2. 实现 com.freedomland.api.core.IModEntry，并在 mod.json 中声明入口类
  3. 使用 mods/build_mod.bat 或你自己的构建脚本打包
  4. 将模组 JAR 放入 mods/，通过启动器选择存档并进入游戏

参考：模组开发指南

4. 常见问题

  • 游戏必须由官方启动器启动（令牌校验）
  • FLML 核心包需为官方签名/标识版本且与游戏版本兼容
  • 模组依赖需在 mod.json 声明并满足版本范围
